hexo主题有什么推荐的吗?

发表于2020-12-17|更新于2020-12-26
|阅读量:

主题选择是个很主观的事情，找一个你觉得顺眼的主题即可。以下是我推荐的几个顺眼的 hexo 主题，demo 为主题示例。

matery

demo

github 地址

卡片式风格的主题，排列和动画都很好看。

butterfly

demo

github 地址

如果你的博客有大量图片的话，推荐这款

fluid

demo

github 地址

material design 的主题，看一眼就知道是不是你的菜

icarus

demo

赛博朋克 2077demo

github 地址

一款清爽的三列式主题，最近还上线了它的赛博朋克 2077 的版本皮肤，非常炫酷

next

demo

github 地址

非常精简的一款主题，黑白配又包含大量留白，简约爱好者必选。

最后

最后的最后，我还是建议，在看过几款 demo 之后，锁定你看得最顺眼的那一款，不要再纠结别的主题好不好看了。博客最重要的是你的内容，主题不要花太多时间去折腾，因为没有内容，主题再花哨也没有意义。用心打磨你的内容，分享你的思考和收获，比任何好看的主题都重要。
